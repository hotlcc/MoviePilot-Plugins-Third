## 影视收藏助手

### 1、简介

作者：hotlcc

主页：[Github](https://github.com/hotlcc) | [Gitee](https://gitee.com/hotlcc)

#### 1.1、功能说明

- 用于将MP中的影视信息收藏到其他媒介，以供随时查阅。
- 模块化设计，目前仅支持收藏到 Linkding，未来会考虑别的介质。
